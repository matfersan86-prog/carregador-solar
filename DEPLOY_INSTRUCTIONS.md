# 🚀 Instruções de Deploy Permanente

Seu site TechBrasil está pronto para ser publicado online! Escolha uma das opções abaixo:

## Opção 1: Vercel (Recomendado - Mais Fácil)

### Passo 1: Criar conta no Vercel
1. Acesse https://vercel.com
2. Clique em "Sign Up"
3. Faça login com GitHub, GitLab ou email

### Passo 2: Fazer Deploy
1. Clique em "New Project"
2. Selecione "Import Git Repository"
3. Cole a URL do seu repositório GitHub
4. Clique em "Deploy"

**Seu site estará online em minutos!**

---

## Opção 2: Netlify

### Passo 1: Criar conta no Netlify
1. Acesse https://netlify.com
2. Clique em "Sign up"
3. Faça login com GitHub

### Passo 2: Fazer Deploy
1. Clique em "New site from Git"
2. Conecte seu repositório GitHub
3. Clique em "Deploy site"

**Pronto! Seu site estará online.**

---

## Opção 3: GitHub Pages (Gratuito)

### Passo 1: Fazer Push para GitHub
```bash
git remote add origin https://github.com/seu-usuario/carregador-solar.git
git branch -M main
git push -u origin main
```

### Passo 2: Ativar GitHub Pages
1. Vá para Settings do repositório
2. Procure por "Pages"
3. Selecione "Deploy from a branch"
4. Escolha "main" como branch
5. Clique em "Save"

**Seu site estará em: https://seu-usuario.github.io/carregador-solar**

---

## Opção 4: Hospedagem Compartilhada (Hostinger, Bluehost, etc)

1. Faça upload de todos os arquivos via FTP
2. Certifique-se de que `index.html` está na raiz
3. Acesse seu domínio

## 🎯 Domínio Personalizado

Após publicar, você pode adicionar um domínio personalizado:

1. **Compre um domínio** em: GoDaddy, Namecheap, Registro.br
2. **Configure o DNS** para apontar para seu host
3. **Ative HTTPS** (geralmente automático)

**Exemplo:** https://www.techbrasil.com.br
