## 🎯 OPÇÃO 1: DEPLOY RÁPIDO (RECOMENDADO) - Vercel

### Passo 1: Preparar o Repositório GitHub

```bash
# Se ainda não tem Git instalado
git init
git add .
git commit -m "Site TechBrasil - Versão Final com 10 Funcionalidades"

# Criar repositório no GitHub
# Acesse: https://github.com/new
# Nome: carregador-solar
# Descrição: Site de vendas do Carregador Solar TechBrasil
```

### Passo 2: Fazer Push para GitHub

```bash
git remote add origin https://github.com/SEU_USUARIO/carregador-solar.git
git branch -M main
git push -u origin main
```

### Passo 3: Deploy no Vercel

1. Acesse: **https://vercel.com**
2. Clique em **"Sign Up"** (ou faça login)
3. Clique em **"New Project"**
4. Selecione **"Import Git Repository"**
5. Cole a URL do seu repositório GitHub
6. Clique em **"Deploy"**

**Pronto! Seu site estará online em 2-3 minutos!**

Você receberá um link como: `https://carregador-solar.vercel.app`

---

## 🎯 OPÇÃO 2: NETLIFY (ALTERNATIVA)

1. Acesse: **https://netlify.com**
2. Clique em **"Sign up"**
3. Conecte sua conta GitHub
4. Clique em **"New site from Git"**
5. Selecione seu repositório
6. Clique em **"Deploy site"**

**Seu site estará online em minutos!**

---

## 🎯 OPÇÃO 3: GitHub Pages (GRATUITO)

1. Vá para **Settings** do seu repositório
2. Procure por **"Pages"**
3. Em **"Source"**, selecione **"Deploy from a branch"**
4. Escolha **"main"** como branch
5. Clique em **"Save"**

**Seu site estará em:** `https://seu-usuario.github.io/carregador-solar`

## 🌐 ADICIONAR DOMÍNIO PERSONALIZADO

Depois de publicar, você pode usar seu próprio domínio!

### Passo 1: Comprar um Domínio

Escolha um registrador:
- **Registro.br** (domínios .com.br)
- **GoDaddy** (domínios internacionais)
- **Namecheap** (barato e confiável)
- **Hostinger** (integrado com hospedagem)

**Exemplo de domínio:** `www.techbrasil.com.br`

### Passo 2: Configurar DNS

No painel do Vercel/Netlify:
1. Vá para **Settings** → **Domains**
2. Adicione seu domínio
3. Copie os registros DNS
4. Cole no painel do seu registrador

**Aguarde 24-48 horas para propagação.**
